# Senior Fullstack Interview Handbook

Статический сайт и исходный Markdown для подготовки к техническому интервью.

## Файлы

- `senior-fullstack-interview-handbook.md` — содержание handbook.
- `index.html` — статическая оболочка: Markdown-рендеринг, навигация, подсветка кода и проверка обновлений.
- `CNAME` — пользовательский домен GitHub Pages.

Для локального просмотра можно открыть `index.html` через любой статический HTTP-сервер.

## Языковые режимы

- Переключатель в шапке сайта: **RU / RU+EN / EN**; выбор сохраняется в localStorage и переживает перезагрузку.
- Markdown остаётся единым исходником, три представления строятся при рендеринге:
  - HTML-комментарии `<!-- lang:rus -->`, `<!-- lang:eng -->`, `<!-- lang:all -->` задают регион для вводных разделов: русские вступление и глоссарий видны в режимах RU и RU+EN, английские — только в EN.
  - В регионе `all` абзацы и списки классифицируются автоматически: есть кириллица → русский блок, нет → английский. Заголовки, разделители, код, Mermaid-схемы и таблицы общие и не дублируются.
