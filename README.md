# Senior Fullstack Interview Handbook

Статический сайт и исходный Markdown для подготовки к техническому интервью.

## Файлы

- `senior-fullstack-interview-handbook.md` — содержание handbook.
- `index.html` — статическая оболочка: Markdown-рендеринг, навигация, подсветка кода и проверка обновлений.
- `CNAME` — пользовательский домен GitHub Pages.

Для локального просмотра можно открыть `index.html` через любой статический HTTP-сервер.

## Языковые режимы

- Переключатель в шапке сайта: **RU / UA / RU+EN / UA+EN / EN**; выбор сохраняется в localStorage и переживает перезагрузку.
- Исходники разделены по языкам: `handbook.rus.md`, `handbook.ukr.md` и `handbook.eng.md`.
  - Режимы RU, UA и EN рендерят соответствующий Markdown-файл.
  - В RU+EN и UA+EN сайт совмещает секции по совпадающим заголовкам `h3`: сначала родной текст, затем общий код или Mermaid-схема, затем английский ответ. Поэтому пример не дублируется и не зависит от эвристики по алфавиту.
  - Каждый украинский абзац обязан содержать хотя бы одну из букв і/ї/є/ґ, а русский текст не должен их содержать — на этом держится автоклассификация.
