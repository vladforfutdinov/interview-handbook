# Senior Fullstack Interview Handbook

Статический сайт и исходный Markdown для подготовки к техническому интервью.

## Файлы

- `handbook.rus.md`, `handbook.ukr.md`, `handbook.eng.md` — исходники handbook по языкам.
- `index.html` — статическая оболочка: Markdown-рендеринг, навигация, подсветка кода и проверка обновлений.
- `CNAME` — пользовательский домен GitHub Pages.

Для локального просмотра можно открыть `index.html` через любой статический HTTP-сервер.

## Языки

- В шапке выбирается базовый язык: **UA / EN / RU**. Его вступление, глоссарий и ответы идут первыми.
- Ниже можно отметить дополнительные языки чекбоксами; для базового языка чекбокс отключён. Порядок дополнительных вариантов всегда **UA → EN → RU**.
- Для каждой совпадающей секции `h3` сайт показывает базовый ответ, один общий блок кода или Mermaid-схему, затем ответы выбранных дополнительных языков. Выбор сохраняется в localStorage и переживает перезагрузку.
