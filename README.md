# Senior Fullstack Interview Handbook

Статический сайт и исходный Markdown для подготовки к техническому интервью.

## Файлы

- `senior-fullstack-interview-handbook.md` — содержание handbook.
- `index.html` — статическая оболочка: Markdown-рендеринг, навигация, подсветка кода и проверка обновлений.
- `CNAME` — пользовательский домен GitHub Pages.

Для локального просмотра можно открыть `index.html` через любой статический HTTP-сервер.

## Языковые режимы

- Переключатель в шапке сайта: **RU / UA / RU+EN / EN**; выбор сохраняется в localStorage и переживает перезагрузку.
- Markdown остаётся единым исходником, представления строятся при рендеринге:
  - HTML-комментарии `<!-- lang:rus -->`, `<!-- lang:ukr -->`, `<!-- lang:eng -->`, `<!-- lang:all -->` задают регион для вводных разделов: русские вступление и глоссарий видны в режимах RU и RU+EN, украинские — только в UA, английские — только в EN.
  - В регионе `all` абзацы и списки классифицируются автоматически: есть украинские буквы (і, ї, є, ґ) → украинский блок; иначе есть кириллица → русский; иначе → английский. Заголовки, разделители, код, Mermaid-схемы и таблицы общие и не дублируются.
  - Каждый украинский абзац обязан содержать хотя бы одну из букв і/ї/є/ґ, а русский текст не должен их содержать — на этом держится автоклассификация.
